== NAME

getpidino - retrieve the pidfd inode number for a given PID

== SYNOPSIS

*getpidino* [*--print-pid*|*-p*] _PID_...

== DESCRIPTION

*getpidino* is a simple command that prints the pidfd inode numbers for all PIDs 
passed to it as arguments.
*getpidino* wraps around the system call *pidfd_open(2)* in order to retrieve a pidfd
that is eventually passed to *fstat(2)* to obtain its inode number.


== OPTIONS

*-p*, *--print-pid*::
Print both the PID and pidfd inode separated by a colon _:_.
